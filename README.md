### 🛠️ Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Flexbox, Grid, Media Queries, Animations
- **JavaScript** — DOM Manipulation, Event Handling

### ✨ Features
- ✅ Responsive Product Grid with multiple items
- ✅ Interactive Product Cards with Hover Effects  
- ✅ Shopping Cart UI with Add/Remove functionality
- ✅ Mobile-First Responsive Design
- ✅ Clean & Modern User Interface

### 🎯 What I Learned
- Building complex layouts using CSS Grid & Flexbox
- JavaScript DOM manipulation for cart functionality
- Creating responsive designs that work on all devices
